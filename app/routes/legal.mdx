# Політика використання кукі-файлів

Ця політика використання кукі-файлів пояснює, як **ITBoost** (далі — "Компанія", "ми") використовує файли кукі та подібні технології для збирання та зберігання інформації під час вашого користування нашим вебсайтом.

<br/>

![Cookies](/Cookies.jpg)

<br/>

#### 1. Що таке кукі-файли?

Кукі-файли — це невеликі текстові файли, що зберігаються на вашому пристрої під час відвідування вебсайтів. Вони можуть використовуватися для збереження інформації про ваші дії на сайті, уподобання та налаштування, щоб покращити ваш досвід користування.

<br/>
<hr/>
<br/>

#### 2. Типи кукі-файлів, які ми використовуємо

Ми використовуємо тільки функціональні кукі-файли, які необхідні для забезпечення нормальної роботи нашого сайту. Ці файли не зберігають персональні дані і не використовуються для збору інформації, яка може ідентифікувати вас як особу. Вони включають:

<br/>

- **Обов'язкові кукі-файли**: Необхідні для правильного функціонування нашого сайту. Вони дозволяють вам переміщатися сайтом і використовувати його функції.
- **Функціональні кукі-файли**: Використовуються для збереження ваших налаштувань, таких як вибір мови або регіону.

<br/>

Ми не використовуємо аналітичні, рекламні чи інші типи кукі-файлів, які можуть збирати або використовувати персональні дані для таргетованої реклами.

<br/>

#### 3. Використання та мета збирання кукі-файлів

Файли кукі допомагають нам:

<br/>

- Забезпечити безперебійну роботу нашого вебсайту;
- Запам'ятати ваші налаштування та вподобання для покращення користувацького досвіду;
- Захистити ваші дані та безпеку під час використання сайту.

<br/>

Ми не передаємо інформацію, що зберігається в кукі-файлах, третім особам, за винятком випадків, передбачених законодавством.

#### 4. Як ви можете керувати кукі-файлами?

Ви можете налаштувати свій браузер так, щоб він блокував або видаляв кукі-файли. Зверніть увагу, що видалення або блокування деяких кукі може вплинути на роботу вебсайту, і деякі функції можуть стати недоступними.

Найбільш поширені способи управління кукі-файлами доступні через налаштування браузера. Інструкції з налаштування можна знайти на офіційних сайтах браузерів:
<br/>
- Google Chrome
- Mozilla Firefox
- Safari
- Microsoft Edge
